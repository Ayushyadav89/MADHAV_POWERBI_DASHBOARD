# 📊 Madhav Store Power BI Dashboard

A professional **Power BI Dashboard** built using retail store sales data to analyze business performance, identify sales trends, and provide actionable insights for decision-making.

---

## 📌 Project Overview

The **Madhav Store Power BI Dashboard** is an interactive business intelligence project that transforms raw retail sales data into meaningful visualizations. It helps stakeholders monitor key performance indicators (KPIs), understand customer purchasing behavior, and analyze profitability across different regions, products, and categories.

This project demonstrates the use of **Power BI** for data cleaning, modeling, visualization, and dashboard design.

---

## 🚀 Features

* 📈 Interactive Sales Dashboard
* 💰 Total Sales, Profit, and Quantity Analysis
* 🛒 Category-wise Sales Performance
* 🌍 Region and State-wise Sales Analysis
* 👥 Customer Segment Insights
* 📅 Monthly and Yearly Sales Trends
* 📦 Product Category Performance
* 🎛️ Dynamic Filters and Slicers
* 📊 KPI Cards for Quick Business Overview

```bash

```

2. Open the `.pbix` file using **Microsoft Power BI Desktop**.

3. Refresh the dataset if required.

4. Explore the interactive dashboard using slicers and filters.
